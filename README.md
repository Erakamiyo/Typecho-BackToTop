# BackToTop

简单 轻量级

随机三款东方样式,刷新页面改变


#【食用方法】
<br>
Download ZIP, 解压
<br>
重命名文件夹为BackToTop
<br>
上传至usr/plugins目录
<br>
登录后台启用即可
<br>
如果主题模板已加载jquery，可以在插件设置内禁用加载jquery
